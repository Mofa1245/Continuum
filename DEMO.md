# Continuum Demo (10–15 sec)

## Step 1
Run:
`npm run continuum -- drift-demo`

## Step 2
Start UI:
`npm run continuum -- ui`

## Step 3
Open:
`http://localhost:3000`

## Step 4
Click:
`Debug Latest Drift`

## What you should see:
- Drift detected (red)
- Prompt change ("as JSON")
- amount: 72 -> "72.00"
- Root cause + fix
